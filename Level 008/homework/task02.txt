Input და output კომპიუტერულ პროგრამებში აღწერს მონაცემების მოძრაობას.

Input (შესავალი მონაცემები) — ეს არის მონაცემები, რომლებსაც პროგრამა იღებს გარე სამყაროდან (მაგალითად, მომხმარებლისგან, ფაილებიდან ან სენსორებიდან) შემდგომი დამუშავების ან ანალიზისათვის.

Output (გამავალი მონაცემები) — ეს არის მონაცემები, რომლებსაც პროგრამა აწვდის გარე სამყაროს (მაგალითად, ეკრანზე, ფაილში ან სხვა სისტემაში) მისი მუშაობის შედეგების სახით.


ნაკადები — Python-ში ნაკადი (stream) წარმოადგენს მონაცემების შესვლის ან გამოსვლის არხს. არსებობს სამი ძირითადი ნაკადი:

1. Standard Input (stdin) — მონაცემების მიღების ძირითადი ნაკადი.

ბრძანება: input()



2. Standard Output (stdout) — მონაცემების გადაცემის ძირითადი ნაკადი (ეკრანზე ან ფაილში).

ბრძანება: print()



3. Standard Error (stderr) — შეცდომების გადასაცემი ნაკადი.

ბრძანება: sys.stderr.write() (ამ ნაკადზე წვდომისთვის საჭიროა sys მოდულის იმპორტი)




Python-ში, სტანდარტული input() და print() ბრძანებები ძირითადად საკმარისია ყოველდღიურ პროგრამებში მონაცემების მიღებისა და გამოშვებისთვის.
