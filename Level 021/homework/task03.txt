### CSS-ის დეკლარაცია, კუთვნილება და მნიშვნელობა

CSS (Cascading Style Sheets) გამოიყენება HTML ელემენტებზე ვიზუალური სტილის განსახორციელებლად. თითოეული CSS-ს წესი შედგება რამდენიმე ძირითადი კომპონენტისგან: **დეკლარაცია**, **კუთვნილება** და **მნიშვნელობა**.

#### 1. **CSS-ის დეკლარაცია**
CSS-ის დეკლარაცია არის ის ნაწილები, რომლებიც განსაზღვრავს, როგორ უნდა გამოიყურებოდეს HTML ელემენტი. დეკლარაციაში შეტანილია **კუთვნილება** (property) და **მნიშვნელობა** (value), რომლებიც განსაზღვრავენ ელემენტის ვიზუალურ თვისებებს.

**დეკლარაციის ფორმატი:**
```
property: value;
```

**მაგალითი:**
```css
color: red;
```
აქ, `color` არის **კუთვნილება**, ხოლო `red` არის **მნიშვნელობა**.

#### 2. **კუთვნილება (Property)**
კუთვნილება განსაზღვრავს, თუ რომელი ასპექტი უნდა შეიცვალოს ელემენტში. მაგალითად, ეს შეიძლება იყოს ელემენტის ფერი, შრიფტი, ან ზომა.

**ზოგიერთი გავრცელებული კუთვნილება:**
- `color` – განსაზღვრავს ტექსტის ფერს.
- `font-size` – განსაზღვრავს შრიფტის ზომას.
- `background-color` – განსაზღვრავს ელემენტის ფონურ ფერს.
- `width` – განსაზღვრავს ელემენტის სიგანეს.
- `height` – განსაზღვრავს ელემენტის სიმაღლეს.

**მაგალითი:**
```css
font-size: 16px;
```
აქ, `font-size` არის **კუთვნილება**, რომელიც განსაზღვრავს შრიფტის ზომას, ხოლო `16px` არის მისი **მნიშვნელობა**.

#### 3. **მნიშვნელობა (Value)**
მნიშვნელობა განსაზღვრავს კონკრეტულ პარამეტრს, რომელსაც უნდა მიეკუთვნოს კუთვნილება. მნიშვნელობები შეიძლება იყოს სხვადასხვა ტიპის: ციფრები, ტექსტები, ფერები და სხვ.

**მნიშვნელობების ტიპები:**
- **ციფრები** (მაგალითად, `12px`, `100%`, `1`).
- **ფერები** (მაგალითად, `red`, `#ff0000`, `rgb(255, 0, 0)`).
- **ტექსტი** (მაგალითად, `solid`, `italic`, `bold`).
- **პროცენტული მნიშვნელობები** (მაგალითად, `50%`, `100%`).

**მაგალითი:**
```css
background-color: blue;
```
აქ, `blue` არის **მნიშვნელობა**, რომელიც განსაზღვრავს ფონურ ფერს.

### მაგალითები:

1. **მაგალითი 1: ტექსტის ფერის შეცვლა**
   ```css
   p {
     color: green; /* color არის კუთვნილება, green არის მნიშვნელობა */
   }
   ```
   - **კუთვნილება**: `color`
   - **მნიშვნელობა**: `green`
   - ამ წესის შედეგად `<p>` ელემენტი გაიღიმებს მწვანედ.

2. **მაგალითი 2: შრიფტის ზომის შეცვლა**
   ```css
   h1 {
     font-size: 24px; /* font-size არის კუთვნილება, 24px არის მნიშვნელობა */
   }
   ```
   - **კუთვნილება**: `font-size`
   - **მნიშვნელობა**: `24px`
   - ეს წესით, ყველა `<h1>` ელემენტისთვის 24px ზომის შრიფტი განისაზღვრება.

3. **მაგალითი 3: ელემენტის ფონური ფერის შეცვლა**
   ```css
   div {
     background-color: yellow; /* background-color არის კუთვნილება, yellow არის მნიშვნელობა */
   }
   ```
   - **კუთვნილება**: `background-color`
   - **მნიშვნელობა**: `yellow`
   - ეს წესით, ყველა `<div>` ელემენტი გახდება ყვითელი ფონით.

### დასკვნა:

- **CSS-ის დეკლარაცია** წარმოადგენს კუთვნილების და მნიშვნელობის ერთობლიობას, რომელიც განსაზღვრავს, როგორ უნდა გამოიყურებოდეს HTML ელემენტები.
- **კუთვნილება** (property) განსაზღვრავს, თუ რა თვისებას უნდა მიეცეს HTML ელემენტს (მაგალითად, ფერი, ზომა, შრიფტი).
- **მნიშვნელობა** (value) განსაზღვრავს ამ თვისების კონკრეტულ მნიშვნელობას (მაგალითად, ფერი იქნება წითელი, ზომა იქნება 16px და ა.შ.).

ეს სამი კომპონენტი ერთმანეთთან მუშაობს იმისათვის, რომ ვებ-გვერდზე ელემენტები ვიზუალურად გამოიყურებოდეს ისე, როგორც დიზაინერი ან დეველოპერი ითხოვს.
