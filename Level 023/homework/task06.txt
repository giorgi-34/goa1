`colspan` და `rowspan` არიან HTML ატრიბუტები, რომლებიც გამოიყენება ცხრილის `<td>` (ჩვეულებრივი უჯრა) ან `<th>` (სვეტის სათაური) ელემენტებში, რათა მათ გაერთიანებას შეძლონ რამდენიმე სვეტი ან სტრიქონი. აქ არის HTML კოდი, სადაც მოტანილია მაგალითები და კომენტარების დახმარებით არის ახსნილი, რას აკეთებენ ისინი:

### HTML კოდი:
```html
<!DOCTYPE html>
<html lang="ka">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>colspan და rowspan</title>
</head>
<body>
    <h1>colspan და rowspan მაგალითები</h1>

    <table border="1">
        <thead>
            <tr>
                <th>დრო</th>
                <th>ორშაბათი</th>
                <th>სამშაბათი</th>
                <th>ოთხშაბათი</th>
            </tr>
        </thead>
        <tbody>
            <tr>
                <td>08:00 - 09:00</td>
                <td colspan="3">გაკვეთილი 1</td> <!-- colspan="3" ნიშნავს რომ ეს უჯრა დაიკავებს 3 სვეტს -->
            </tr>
            <tr>
                <td rowspan="2">09:00 - 11:00</td> <!-- rowspan="2" ნიშნავს რომ ეს უჯრა დაიკავებს 2 სტრიქონს -->
                <td>გაკვეთილი 2</td>
                <td>გაკვეთილი 2</td>
                <td>გაკვეთილი 2</td>
            </tr>
            <tr>
                <td>გაკვეთილი 3</td>
                <td>გაკვეთილი 3</td>
                <td>გაკვეთილი 3</td>
            </tr>
        </tbody>
    </table>

</body>
</html>
```

### კომენტარები:
1. **`colspan="3"`**: 
   - ეს ატრიბუტი გამოიყენება იმისთვის, რომ `<td>` ან `<th>` ელემენტი დაიკავოს რამდენიმე სვეტი ერთ ხაზზე. მაგალითად, `<td colspan="3">გაკვეთილი 1</td>` ნიშნავს, რომ "გაკვეთილი 1" აიღებს 3 სვეტს (ორშაბათს, სამშაბათს და ოთხშაბათს).
   - **ასევე**: თუ სვეტის გამოფენა არ არის საჭირო (თუმცა ამ შემთხვევაში სვეტები გამოჩნდება, რადგან აქ 3 სვეტი უნდა დაიკავოს), შეგიძლიათ `colspan`-ი გაზარდოთ.

2. **`rowspan="2"`**:
   - ეს ატრიბუტი გამოიყენება იმისთვის, რომ `<td>` ან `<th>` ელემენტი დაიკავოს რამდენიმე სტრიქონი (ხაზი). მაგალითად, `<td rowspan="2">09:00 - 11:00</td>` ნიშნავს, რომ "09:00 - 11:00" მნიშვნელობა გაფართოვდება ორი სტრიქონის მასშტაბით. ეს იძლევა მასშტაბურ განლაგებას, როდესაც უნდა ვაჩვენოთ, რომ მონაცემები იკავებენ ზედა და ქვედა სტრიქონებს.
   - **ამას** ავსებს იმ ფაქტი, რომ ეს სტრიქონი მხოლოდ ერთია, მაგრამ იგი ეწევა ორივე სტრიქონს (ის გავლენას ახდენს მათ). 

### შედეგი:
1. პირველი სტრიქონი (08:00 - 09:00) ექვემდებარება `colspan="3"`, რის შედეგადაც "გაკვეთილი 1" აიღებს 3 სვეტს.
2. მეორე სტრიქონი (09:00 - 11:00) გამოყენებულია `rowspan="2"`, ასე რომ, დრო "09:00 - 11:00" ამ სტრიქონს გადაფარავს და იგი დაიკავებს ორივე სტრიქონს (სამი დღის განმავლობაში).
