### Hexadecimal Color Model-ის შესახებ

Hexadecimal color model (თექვსმეტობითი ფერების მოდელი) გამოიყენება კომპიუტერული სისტემებში ფერების დასახატად. ეს მოდელი იყენებს 16-კონწუპს სისტემას, სადაც თითოეულ ფერს ხასიათდება სამი ძირითადი კომპონენტი:

1. **Red (წითელი)**
2. **Green (მწვანე)**
3. **Blue (ლურჯი)**

ამ სამ კომპონენტს ერთად ქმნიან RGB (Red, Green, Blue) ფერის მოდელს, რომელიც აერთიანებს თითოეულ კომპონენტს hex (თექვსმეტობითი) მნიშვნელობებით.

### Hexadecimal (Hex) ფერის კოდები

- Hexadecimal სისტემა იყენებს 16 სიმბოლოს: 0-9 და A-F.
    - **0-9**: ციფრები.
    - **A-F**: ასოები, რომლებიც წარმოადგენენ მნიშვნელობებს 10-15.
  
ფერის კოდი გამოიყურება როგორც ექვსნიშნა ციფერი: **#RRGGBB**, სადაც:

- **RR** - წითელი კომპონენტი (Red)
- **GG** - მწვანე კომპონენტი (Green)
- **BB** - ლურჯი კომპონენტი (Blue)

### Hexadecimal კოდის მაგალითები:

1. **#FF0000**: 
   - **FF** (255) წითელი - მაქსიმალური სიძლიერე.
   - **00** (0) მწვანე - არ არის მწვანე.
   - **00** (0) ლურჯი - არ არის ლურჯი.
   - **ფერი**: წითელი

2. **#00FF00**:
   - **00** (0) წითელი - არ არის წითელი.
   - **FF** (255) მწვანე - მაქსიმალური სიძლიერე.
   - **00** (0) ლურჯი - არ არის ლურჯი.
   - **ფერი**: მწვანე

3. **#0000FF**:
   - **00** (0) წითელი - არ არის წითელი.
   - **00** (0) მწვანე - არ არის მწვანე.
   - **FF** (255) ლურჯი - მაქსიმალური სიძლიერე.
   - **ფერი**: ლურჯი

4. **#FFFFFF**:
   - **FF** (255) წითელი - მაქსიმალური სიძლიერე.
   - **FF** (255) მწვანე - მაქსიმალური სიძლიერე.
   - **FF** (255) ლურჯი - მაქსიმალური სიძლიერე.
   - **ფერი**: თეთრი

5. **#000000**:
   - **00** (0) წითელი - არ არის წითელი.
   - **00** (0) მწვანე - არ არის მწვანე.
   - **00** (0) ლურჯი - არ არის ლურჯი.
   - **ფერი**: შავი

### Hexadecimal Color-ის შექმნის პროცესი:

1. **განსაზღვრე ძირითადი ფერები**: წითელი, მწვანე და ლურჯი.
2. **დაიწერე თითოეული კომპონენტის სტრუქტურა**: თითოეული კომპონენტი (წითელი, მწვანე, ლურჯი) მიიღებს მნიშვნელობას 00-დან FF-მდე (16-კონწუპსი ციფრები).
3. **კოდის ფორმატირება**: 3-დან 6-ნიშნა Hexadecimal კოდებით.

### ვიზუალური ნახაზი:

```
----------------------------------------
|           Hexadecimal Color Model    |
----------------------------------------
|      R (Red)     G (Green)   B (Blue)|
|       00-FF      00-FF       00-FF  |
----------------------------------------
|   Example Colors:                    |
|   #FF0000 -> Red                      |
|   #00FF00 -> Green                    |
|   #0000FF -> Blue                     |
|   #FFFFFF -> White                    |
|   #000000 -> Black                    |
----------------------------------------
```

- **#FF0000** = Red (წითელი)
- **#00FF00** = Green (მწვანე)
- **#0000FF** = Blue (ლურჯი)
- **#FFFFFF** = White (თეთრი)
- **#000000** = Black (შავი)

### Hexadecimal ფერის მოდელი მარტივად გარდაქმნის RGB ფერებს თექვსმეტობითი სისტემაში, რაც საშუალებას გვაძლევს ვმართოთ ფერები კომპიუტერულ სისტემებში, რომლებსაც აქვთ კონკრეტული ფერის ხარისხი.
