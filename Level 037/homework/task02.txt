აქ არის 50-ვე კითხვის პასუხი:

1. print(type("Hello, World!")) → <class 'str'>


2. Python-ის სამი მონაცემთა ტიპი: int, float, str


3. range() ფუნქცია ქმნის რიცხვების სერიას.


4. Python-ში კომენტარის დასაწერად ვიყენებთ # სიმბოლოს.


5. ცვლადი ინახავს მონაცემებს, რომლებსაც ვიყენებთ კოდში.


6. Python-ში ფუნქციის შესაქმნელად ვიყენებთ def-ს.


7. Keyword (საკვანძო სიტყვა) არის სიტყვა, რომელიც დაცულია Python-ის მიერ და არ შეიძლება გამოყენებულ იქნას როგორც ცვლადი.


8. print(2 > 3) → False


9. ლუწი და კენტი რიცხვების გასარკვევად ვიყენებთ % ოპერატორს.


10. print('Hello ' + 'World') → Hello World


11. print(4 % 2) → 0


12. ცვლადი შეიძლება შეიცვალოს კოდის შესრულებისას.


13. print(10 == 10) → True


14. print(5 ** 3) → 125


15. print('Hello'[1]) → 'e'


16. print(list(range(2, 10, 2))) → [2, 4, 6, 8]


17. while ციკლი მუშაობს მანამ, სანამ პირობა მართალია.


18. while ციკლი გამოიყენება, როცა არ ვიცით გამეორების რაოდენობა.


19. boolean ტიპს აქვს ორი მნიშვნელობა: True და False.


20. while ციკლი იყენებს პირობას და მუშაობს მანამ, სანამ ის მართალია.


21. for ციკლი გამოიყენება მონაცემთა სტრუქტურაში ელემენტების გადასაკრავად.


22. boolean ტიპის მონაცემები შეიძლება იყოს True ან False.


23. boolean(0) აბრუნებს False-ს.


24. boolean([]) აბრუნებს False-ს, რადგან ცარიელი სია არის False.


25. 4-ელემენტიან სიაში ინდექსების დიაპაზონია 0-დან 3-მდე.


26. HTML ნიშნავს HyperText Markup Language-ს.


27. <head> თეგი შეიცავს მეტამონაცემებს, სკრიპტებს და სტილის ფაილებს.


28. Hyperlink არის ბმული, რომელიც გადაამისამართებს სხვა გვერდზე ან რესურსზე.


29. <a> თეგი გამოიყენება ბმულების (hyperlink) შესაქმნელად.


30. <meta> თეგი გამოიყენება მეტამონაცემების განსაზღვრისთვის, როგორიცაა კოდირების ტიპი.


31. title თეგი განსაზღვრავს ვებგვერდის სათაურს.


32. საიტის მთავარი სათაური განსაზღვრავს ვებგვერდის მთავარ კონტენტს.


33. <h1> თეგი წარმოადგენს ყველაზე დიდ სათაურს HTML-ში.


34. checkbox არის მონიშვნის ველი, რომელიც მომხმარებელს საშუალებას აძლევს აირჩიოს ერთი ან რამდენიმე ვარიანტი.


35. css-ში background-color განსაზღვრავს ფონის ფერს.


36. ფონის სურათის ჩასასმელად ვიყენებთ background-image-ს.


37. ბმულის ფერის შესაცვლელად ვიყენებთ color ატრიბუტს css-ში.


38. <em> თეგი გამოიყენება ტექსტის გამოკვეთილად გამოსაჩენად (ხშირად დახრილი სტილით).


39. <form> თეგი გამოიყენება ფორმების შესაქმნელად, რომლებიც შეიცავს მომხმარებლის შეყვანის ველებს.


40. ტექსტური შეყვანის ველი იქმნება <input type="text">-ით.


41. პაროლის შეყვანის ველი იქმნება <input type="password">-ით.


42. dropdown მენიუ იქმნება <select> თეგით.


43. <nav> თეგი გამოიყენება ნავიგაციის ბლოკის განსაზღვრისთვის.


44. <main> თეგი განსაზღვრავს ვებსაიტის ძირითად კონტენტს.


45. HTML დოკუმენტი იწყება <!DOCTYPE html> დეკლარაციით.


46. <input> ელემენტი გამოიყენება მონაცემების შეყვანისთვის ფორმებში.


47. <img> თეგი გამოიყენება სურათების ჩასასმელად ვებგვერდზე.


48. <video> თეგი გამოიყენება ვიდეოების ჩასასმელად ვებგვერდზე.


49. <audio> თეგი გამოიყენება აუდიო ფაილების ჩასასმელად.
