**Dictionary** (ლექსიკონი) განსხვავდება აქამდე ნასწავლი კოლექციებისგან, როგორიცაა **list** (სია) და **tuple** (კორტეჟი), რადგან ის **Key-Value** (გასაღები-ღირებულება) წყვილებით ინახავს მონაცემებს, ნაცვლად იმისა, რომ მონაცემები მხოლოდ ინდექსებით იყოს მისაწვდომი.  

### **Key, Value, Key-Value Pair**  
- **Key (გასაღები)** – უნიკალური იდენტიფიკატორი, რომელიც გამოიყენება მონაცემის (value) მოსაძებნად. გასაღები უნდა იყოს უნიკალური და შეუცვლელი (immutable), მაგალითად: string, number, tuple.  
- **Value (ღირებულება)** – გასაღების შესაბამისი მნიშვნელობა, რომელიც შეიძლება იყოს ნებისმიერი ტიპის მონაცემი (string, number, list, dictionary და ა.შ.).  
- **Key-Value Pair (გასაღები-ღირებულების წყვილი)** – ორი ელემენტისგან შემდგარი ერთეული, სადაც გასაღები უნიკალურად იდენტიფიცირებს ღირებულებას.  

### **Dictionary-ის უპირატესობა**  
Dictionary საშუალებას გვაძლევს, უფრო სწრაფად ვიპოვოთ და შევცვალოთ მონაცემები გასაღების გამოყენებით, მაშინ როცა **list**-ში ან **tuple**-ში მონაცემებზე წვდომა ხდება ინდექსით, რაც ზოგჯერ ნაკლებად მოსახერხებელია.  

### **მაგალითი**  
```python
# Dictionary-ს შექმნა
student = {
    "name": "გიორგი",
    "age": 21,
    "grade": "A"
}

# Dictionary-დან მონაცემის მიღება key-ს გამოყენებით
print(student["name"])  # გამოიტანს: გიორგი
print(student["age"])   # გამოიტანს: 21
```
აქ **"name"**, **"age"**, და **"grade"** გასაღებებია, ხოლო მათი მნიშვნელობები **"გიორგი"**, **21** და **"A"** შესაბამისი ღირებულებები.
