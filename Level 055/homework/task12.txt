рЃЎрЃљрЃарЃњрЃў, рЃЏрЃЮрЃЊрЃў рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџрЃў CSS Grid рЃЎрЃБрЃЌрЃЋрЃюрЃўрЃџрЃћрЃЉрЃљ рЃЏрЃљрЃарЃбрЃўрЃЋрЃљрЃЊ рЃЊрЃљ рЃњрЃљрЃАрЃљрЃњрЃћрЃЉрЃљрЃЊ рЃљрЃЋрЃ«рЃАрЃюрЃљрЃЌ рЃерЃћрЃюрЃА рЃАрЃўрЃбрЃДрЃЋрЃћрЃЉрЃўрЃЌ, рЃарЃЮрЃЏ рЃњрЃљрЃњрЃўрЃљрЃЊрЃЋрЃўрЃџрЃЊрЃћрЃА рЃгрЃљрЃарЃЏрЃЮрЃЊрЃњрЃћрЃюрЃљ рЃЊрЃљ рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљрЃф:

---

### **12) рЃљрЃ«рЃАрЃюрЃљ рЃЎрЃБрЃЌрЃЋрЃюрЃўрЃџрЃћрЃЉрЃћрЃЉрЃўрЃА:**

---

### РюЁ **1. `display`**
­ЪЊї **рЃарЃљрЃА рЃљрЃЎрЃћрЃЌрЃћрЃЉрЃА:** рЃћрЃБрЃЉрЃюрЃћрЃЉрЃљ рЃЉрЃарЃљрЃБрЃќрЃћрЃарЃА, рЃарЃЮрЃњрЃЮрЃа рЃБрЃюрЃЊрЃљ рЃЏрЃЮрЃўрЃЦрЃфрЃћрЃА рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃў РђЊ рЃЕрЃЋрЃћрЃБрЃџрЃћрЃЉрЃарЃўрЃЋрЃўрЃЋрЃўрЃЌ, рЃЉрЃџрЃЮрЃЎрЃљрЃЊ, рЃ«рЃљрЃќрЃќрЃћ, рЃљрЃю grid-рЃљрЃЊ, flex-рЃљрЃЊ рЃЊрЃљ рЃљ.рЃе.

­ЪћИ **Grid-рЃерЃў рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃАрЃљрЃА:**  
```css
.container {
  display: grid;
}
```
РъА№ИЈ рЃћрЃА рЃюрЃўрЃерЃюрЃљрЃЋрЃА, рЃарЃЮрЃЏ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃў рЃ«рЃЊрЃћрЃЉрЃљ Grid рЃЎрЃЮрЃюрЃбрЃћрЃўрЃюрЃћрЃарЃў рЃЊрЃљ рЃерЃўрЃњрЃюрЃўрЃЌ рЃљрЃарЃАрЃћрЃЉрЃБрЃџ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃА Grid рЃАрЃўрЃАрЃбрЃћрЃЏрЃўрЃЌ рЃљрЃџрЃљрЃњрЃћрЃЉрЃА.

---

### РюЁ **2. `grid-template-rows`**
­ЪЊї **рЃарЃљрЃА рЃљрЃЎрЃћрЃЌрЃћрЃЉрЃА:** рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃљрЃЋрЃА **рЃарЃљрЃЏрЃЊрЃћрЃюрЃў рЃарЃўрЃњрЃў** (row) рЃБрЃюрЃЊрЃљ рЃ░рЃЦрЃЮрЃюрЃЊрЃћрЃА grid-рЃА рЃЊрЃљ рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџрЃўрЃА рЃАрЃўрЃЏрЃљрЃдрЃџрЃћрЃА.

­ЪћИ рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃљрЃЊ:
```css
grid-template-rows: 100px 200px;
```
РъА№ИЈ рЃћрЃА рЃЦрЃЏрЃюрЃўрЃА 2 рЃарЃўрЃњрЃА: рЃърЃўрЃарЃЋрЃћрЃџрЃў 100px рЃАрЃўрЃЏрЃљрЃдрЃџрЃўрЃА, рЃЏрЃћрЃЮрЃарЃћ 200px-рЃўрЃА.

---

### РюЁ **3. `grid-template-columns`**
­ЪЊї **рЃарЃљрЃА рЃљрЃЎрЃћрЃЌрЃћрЃЉрЃА:** рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃљрЃЋрЃА **рЃарЃљрЃЏрЃЊрЃћрЃюрЃў рЃАрЃЋрЃћрЃбрЃў** (column) рЃБрЃюрЃЊрЃљ рЃ░рЃЦрЃЮрЃюрЃЊрЃћрЃА рЃЊрЃљ рЃарЃљ рЃАрЃўрЃњрЃљрЃюрЃћ рЃўрЃДрЃЮрЃА рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџрЃўрЃА.

­ЪћИ рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃљрЃЊ:
```css
grid-template-columns: 1fr 2fr;
```
РъА№ИЈ рЃћрЃА рЃюрЃўрЃерЃюрЃљрЃЋрЃА 2 рЃАрЃЋрЃћрЃбрЃА: рЃърЃўрЃарЃЋрЃћрЃџрЃў 1 рЃгрЃўрЃџрЃў, рЃЏрЃћрЃЮрЃарЃћ 2 рЃгрЃўрЃџрЃў (рЃљрЃЏрЃўрЃбрЃЮрЃЏ рЃЏрЃћрЃЮрЃарЃћ рЃЮрЃарЃ»рЃћрЃа рЃцрЃљрЃарЃЌрЃЮрЃљ).

---

### РюЁ **4. `grid-row-start`**
­ЪЊї **рЃарЃљрЃА рЃљрЃЎрЃћрЃЌрЃћрЃЉрЃА:** рЃљрЃЏрЃЉрЃЮрЃЉрЃА, **рЃарЃЮрЃЏрЃћрЃџрЃў рЃ«рЃљрЃќрЃўрЃЊрЃљрЃю** рЃБрЃюрЃЊрЃљ рЃЊрЃљрЃўрЃгрЃДрЃЮрЃА рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃў рЃЋрЃћрЃарЃбрЃўрЃЎрЃљрЃџрЃБрЃарЃљрЃЊ (рЃќрЃћрЃЏрЃЮрЃЊрЃљрЃю рЃЦрЃЋрЃћрЃЏрЃЮрЃЌ).

­ЪћИ рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃљрЃЊ:
```css
grid-row-start: 2;
```
РъА№ИЈ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃў рЃўрЃгрЃДрЃћрЃЉрЃљ **рЃЏрЃћрЃЮрЃарЃћ рЃ░рЃЮрЃарЃўрЃќрЃЮрЃюрЃбрЃљрЃџрЃБрЃа рЃ«рЃљрЃќрЃўрЃЊрЃљрЃю**.

---

### РюЁ **5. `grid-row-end`**
­ЪЊї **рЃарЃљрЃА рЃљрЃЎрЃћрЃЌрЃћрЃЉрЃА:** рЃљрЃЏрЃЉрЃЮрЃЉрЃА, рЃАрЃљрЃЊ рЃЏрЃЌрЃљрЃЋрЃарЃЊрЃћрЃЉрЃљ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃў Рђћ рЃљрЃюрЃБ **рЃарЃЮрЃЏрЃћрЃџ рЃ«рЃљрЃќрЃљрЃЏрЃЊрЃћ рЃБрЃюрЃЊрЃљ рЃњрЃљрЃЋрЃўрЃЊрЃћрЃА** рЃЋрЃћрЃарЃбрЃўрЃЎрЃљрЃџрЃБрЃарЃљрЃЊ.

­ЪћИ рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃљрЃЊ:
```css
grid-row-end: 4;
```
РъА№ИЈ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃў рЃЏрЃўрЃЊрЃўрЃА **рЃЏрЃћрЃЮрЃЌрЃ«рЃћ рЃ«рЃљрЃќрЃљрЃЏрЃЊрЃћ**, рЃљрЃюрЃБ рЃўрЃЎрЃљрЃЋрЃћрЃЉрЃА рЃарЃўрЃњрЃћрЃЉрЃА 2-рЃЊрЃљрЃю 3-рЃЏрЃЊрЃћ (рЃЌрЃБ рЃЊрЃљрЃгрЃДрЃћрЃЉрЃљ 2-рЃќрЃћрЃљ).

---

### РюЁ **6. `grid-column-start`**
­ЪЊї **рЃарЃљрЃА рЃљрЃЎрЃћрЃЌрЃћрЃЉрЃА:** рЃљрЃЏрЃЉрЃЮрЃЉрЃА, **рЃарЃЮрЃЏрЃћрЃџрЃў рЃ«рЃљрЃќрЃўрЃЊрЃљрЃю** рЃБрЃюрЃЊрЃљ рЃЊрЃљрЃўрЃгрЃДрЃЮрЃА рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃў рЃ░рЃЮрЃарЃўрЃќрЃЮрЃюрЃбрЃљрЃџрЃБрЃарЃљрЃЊ (рЃЏрЃљрЃарЃфрЃ«рЃюрЃўрЃЊрЃљрЃю рЃЏрЃљрЃарЃ»рЃЋрЃюрЃўрЃЋ).

­ЪћИ рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃљрЃЊ:
```css
grid-column-start: 1;
```
РъА№ИЈ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃў рЃўрЃгрЃДрЃћрЃЉрЃљ **рЃърЃўрЃарЃЋрЃћрЃџрЃў рЃЋрЃћрЃарЃбрЃўрЃЎрЃљрЃџрЃБрЃарЃў рЃ«рЃљрЃќрЃўрЃЊрЃљрЃю**.

---

### РюЁ **7. `grid-column-end`**
­ЪЊї **рЃарЃљрЃА рЃљрЃЎрЃћрЃЌрЃћрЃЉрЃА:** рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃљрЃЋрЃА, рЃАрЃљрЃЊ рЃЏрЃЌрЃљрЃЋрЃарЃЊрЃћрЃЉрЃљ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃў рЃ░рЃЮрЃарЃўрЃќрЃЮрЃюрЃбрЃљрЃџрЃБрЃарЃљрЃЊ Рђћ рЃљрЃюрЃБ **рЃарЃЮрЃЏрЃћрЃџ рЃ«рЃљрЃќрЃљрЃЏрЃЊрЃћ рЃБрЃюрЃЊрЃљ рЃњрЃљрЃўрЃГрЃўрЃЏрЃЮрЃА**.

­ЪћИ рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃљрЃЊ:
```css
grid-column-end: 3;
```
РъА№ИЈ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃў рЃўрЃЎрЃљрЃЋрЃћрЃЉрЃА рЃАрЃЋрЃћрЃбрЃћрЃЉрЃА 1-рЃЊрЃљрЃю 2-рЃЏрЃЊрЃћ (рЃЌрЃБ рЃЊрЃљрЃгрЃДрЃћрЃЉрЃљ 1-рЃќрЃћрЃљ).

---

### РюЁ **8. `grid-area`**
­ЪЊї **рЃарЃљрЃА рЃљрЃЎрЃћрЃЌрЃћрЃЉрЃА:** рЃљрЃћрЃарЃЌрЃўрЃљрЃюрЃћрЃЉрЃА `row-start`, `row-end`, `column-start`, `column-end` рЃћрЃарЃЌ рЃ«рЃљрЃќрЃерЃў.

­ЪћИ рЃцрЃЮрЃарЃЏрЃљрЃбрЃў:
```css
grid-area: row-start / column-start / row-end / column-end;
```

­ЪћИ рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў:
```css
grid-area: 1 / 2 / 3 / 4;
```
РъА№ИЈ рЃўрЃгрЃДрЃћрЃЉрЃљ рЃърЃўрЃарЃЋрЃћрЃџ рЃарЃўрЃњрЃерЃў, рЃЏрЃћрЃЮрЃарЃћ рЃАрЃЋрЃћрЃбрЃерЃў рЃЊрЃљ рЃЏрЃЌрЃљрЃЋрЃарЃЊрЃћрЃЉрЃљ рЃЏрЃћрЃАрЃљрЃЏрЃћ рЃарЃўрЃњрЃерЃў, рЃЏрЃћрЃЮрЃЌрЃ«рЃћ рЃАрЃЋрЃћрЃбрЃерЃў.

---

рЃњрЃўрЃюрЃЊрЃљ рЃЏрЃљрЃарЃбрЃўрЃЋрЃў рЃЋрЃўрЃќрЃБрЃљрЃџрЃБрЃарЃў рЃАрЃЦрЃћрЃЏрЃљ рЃљрЃю рЃърЃљрЃбрЃљрЃарЃљ рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў, рЃАрЃљрЃЊрЃљрЃф рЃДрЃЋрЃћрЃџрЃљрЃцрЃћрЃарЃА рЃћрЃарЃЌрЃљрЃЊ рЃњрЃљрЃЏрЃЮрЃЋрЃўрЃДрЃћрЃюрЃћрЃЉрЃЌ? Рюе­ЪЊљ
