рЃАрЃБрЃърЃћрЃа рЃерЃћрЃЎрЃўрЃЌрЃ«рЃЋрЃљрЃљ! рЃЏрЃЮрЃЊрЃў рЃЏрЃљрЃарЃбрЃўрЃЋрЃљрЃЊ рЃљрЃњрЃўрЃ«рЃАрЃюрЃў:

---

### РюЁ **рЃарЃљрЃбрЃЮрЃЏ рЃљрЃарЃўрЃА `grid-area`-рЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ рЃБрЃЏрЃ»рЃЮрЃЉрЃћрЃАрЃў?**

­Ъћ╣ **1. рЃБрЃцрЃарЃЮ рЃЏрЃЮрЃЎрЃџрЃћ рЃЊрЃљ рЃЎрЃЮрЃЏрЃърЃљрЃЦрЃбрЃБрЃарЃўрЃљ**  
`grid-area` рЃљрЃћрЃарЃЌрЃўрЃљрЃюрЃћрЃЉрЃА 4 рЃфрЃљрЃџ-рЃфрЃљрЃџрЃЎрЃћ рЃЎрЃБрЃЌрЃЋрЃюрЃўрЃџрЃћрЃЉрЃљрЃА (`grid-row-start`, `grid-column-start`, `grid-row-end`, `grid-column-end`) рЃћрЃарЃЌ рЃ«рЃљрЃќрЃерЃў.  
рЃћрЃА рЃ«рЃЊрЃўрЃА рЃЎрЃЮрЃЊрЃА рЃЉрЃћрЃЋрЃарЃљрЃЊ рЃЏрЃЮрЃЎрЃџрЃћрЃЊ рЃЊрЃљ рЃАрЃБрЃцрЃЌрЃљрЃЊ.

­ЪћИ рЃЏрЃљрЃњ:  
```css
/* рЃњрЃарЃФрЃћрЃџрЃў рЃЋрЃћрЃарЃАрЃўрЃљ */
.item {
  grid-row-start: 1;
  grid-column-start: 2;
  grid-row-end: 3;
  grid-column-end: 4;
}
```

­ЪЉЅ рЃўрЃњрЃўрЃЋрЃћ рЃарЃљрЃЏ, рЃЏрЃЮрЃЎрЃџрЃћрЃЊ:
```css
.item {
  grid-area: 1 / 2 / 3 / 4;
}
```

---

­Ъћ╣ **2. рЃњрЃљрЃЎрЃЋрЃћрЃЌрЃўрЃџрЃћрЃЉрЃерЃў рЃЊрЃљ рЃърЃарЃЮрЃћрЃЦрЃбрЃћрЃЉрЃерЃў рЃЎрЃЮрЃЊрЃўрЃА рЃгрЃљрЃЎрЃўрЃЌрЃ«рЃЋрЃљ рЃЊрЃљ рЃЏрЃљрЃарЃЌрЃЋрЃљ рЃБрЃцрЃарЃЮ рЃЏрЃљрЃарЃбрЃўрЃЋрЃЊрЃћрЃЉрЃљ**  
рЃарЃЮрЃфрЃљ рЃЉрЃћрЃЋрЃарЃў рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃў рЃњрЃДрЃљрЃЋрЃА Grid-рЃерЃў, рЃћрЃарЃЌ рЃ«рЃљрЃќрЃќрЃћ рЃЊрЃљрЃгрЃћрЃарЃўрЃџрЃў `grid-area` рЃњрЃћрЃ«рЃЏрЃљрЃарЃћрЃЉрЃљ рЃБрЃЎрЃћрЃЌ рЃЊрЃљрЃўрЃюрЃљрЃ«рЃЮ, рЃЌрЃБ рЃАрЃљрЃЊ рЃўрЃгрЃДрЃћрЃЉрЃљ рЃЊрЃљ рЃЏрЃЌрЃљрЃЋрЃарЃЊрЃћрЃЉрЃљ рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџрЃў рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃў.

---

­Ъћ╣ **3. рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ рЃАрЃљрЃ«рЃћрЃџрЃўрЃљрЃюрЃў Grid Template-рЃћрЃЉрЃЌрЃљрЃюрЃљрЃф**  
рЃЌрЃБ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉ рЃАрЃљрЃ«рЃћрЃџрЃћрЃЉрЃА Grid Layout-рЃерЃў (`grid-template-areas`), рЃЏрЃљрЃерЃўрЃю `grid-area` рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃА рЃЎрЃЮрЃюрЃЎрЃарЃћрЃбрЃБрЃџ рЃАрЃљрЃ«рЃћрЃџрЃгрЃЮрЃЊрЃћрЃЉрЃБрЃџ рЃљрЃЊрЃњрЃўрЃџрЃћрЃЉрЃќрЃћ рЃњрЃљрЃюрЃљрЃЌрЃљрЃЋрЃАрЃћрЃЉ:

```css
.container {
  display: grid;
  grid-template-areas:
    "header header"
    "sidebar main";
}

.header {
  grid-area: header;
}
```

---

### ­Ъћџ рЃЊрЃљрЃАрЃЎрЃЋрЃюрЃљ:
**`grid-area` = рЃюрЃљрЃЎрЃџрЃћрЃЉрЃў рЃЎрЃЮрЃЊрЃў + рЃЏрЃћрЃбрЃў рЃАрЃўрЃАрЃБрЃцрЃЌрЃљрЃЋрЃћ + рЃБрЃцрЃарЃЮ рЃАрЃгрЃарЃљрЃцрЃў рЃЏрЃБрЃерЃљрЃЮрЃЉрЃљ.**  
рЃерЃћрЃАрЃљрЃюрЃўрЃерЃюрЃљрЃЋрЃўрЃљ рЃњрЃљрЃюрЃАрЃљрЃЎрЃБрЃЌрЃарЃћрЃЉрЃўрЃЌ рЃЏрЃљрЃерЃўрЃю, рЃарЃЮрЃфрЃљ рЃќрЃБрЃАрЃбрЃљрЃЊ рЃўрЃфрЃў рЃАрЃљрЃЊ рЃњрЃўрЃюрЃЊрЃљ рЃарЃЮрЃЏ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃў рЃњрЃљрЃюрЃЌрЃљрЃЋрЃАрЃЊрЃћрЃА.

---

рЃњрЃўрЃюрЃЊрЃљ рЃърЃљрЃбрЃљрЃарЃљ рЃАрЃљрЃЏрЃБрЃерЃљрЃЮ рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў рЃарЃЮрЃЏ рЃДрЃЋрЃћрЃџрЃљрЃцрЃћрЃарЃў рЃћрЃарЃЌрЃљрЃЊ рЃЋрЃюрЃљрЃ«рЃЮрЃЌ рЃЏрЃЮрЃЦрЃЏрЃћрЃЊрЃћрЃЉрЃљрЃерЃў? ­Ъўі­Ъњ╗
