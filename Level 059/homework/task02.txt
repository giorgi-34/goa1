JavaScript (JS) рЃљрЃарЃўрЃА рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃарЃћрЃЉрЃўрЃА рЃћрЃюрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃАрЃљрЃф рЃЋрЃћрЃЉрЃњрЃЋрЃћрЃарЃЊрЃћрЃЉрЃќрЃћ рЃЋрЃўрЃДрЃћрЃюрЃћрЃЉрЃЌ **рЃўрЃюрЃбрЃћрЃарЃљрЃЦрЃбрЃўрЃБрЃџрЃЮрЃЉрЃўрЃА** рЃЊрЃљрЃАрЃљрЃЏрЃљрЃбрЃћрЃЉрЃџрЃљрЃЊ.

---

### ­ЪЊЇ **рЃАрЃљрЃЊ рЃЋрЃўрЃДрЃћрЃюрЃћрЃЉрЃЌ JavaScript-рЃА?**

* **рЃЋрЃћрЃЉрЃАрЃљрЃўрЃбрЃћрЃЉрЃќрЃћ** Рђћ HTML рЃЊрЃљ CSS-рЃўрЃА рЃњрЃЋрЃћрЃарЃЊрЃўрЃЌ, рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃљрЃЊ `<script>` рЃбрЃћрЃњрЃўрЃЌ рЃљрЃю рЃфрЃљрЃџрЃЎрЃћ `.js` рЃцрЃљрЃўрЃџрЃерЃў
* **рЃЉрЃарЃљрЃБрЃќрЃћрЃарЃерЃў** Рђћ рЃЏрЃљрЃА рЃўрЃДрЃћрЃюрЃћрЃЉрЃА Google Chrome, Firefox, Safari рЃЊрЃљ рЃАрЃ«рЃЋрЃљ рЃЉрЃарЃљрЃБрЃќрЃћрЃарЃћрЃЉрЃў
* **рЃЋрЃћрЃЉрЃАрЃћрЃарЃЋрЃћрЃарЃћрЃЉрЃќрЃћ** Рђћ Node.js-рЃўрЃА рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃўрЃЌ (рЃБрЃцрЃарЃЮ рЃАрЃўрЃдрЃарЃЏрЃўрЃАрЃћрЃБрЃџ рЃЊрЃЮрЃюрЃћрЃќрЃћ)

---

### РЮЊ **рЃарЃљрЃбрЃЮрЃЏ рЃЋрЃўрЃДрЃћрЃюрЃћрЃЉрЃЌ JavaScript-рЃА?**

JavaScript-рЃА рЃЋрЃўрЃДрЃћрЃюрЃћрЃЉрЃЌ рЃЋрЃћрЃЉрЃАрЃљрЃўрЃбрЃќрЃћ:

| рЃЏрЃўрЃќрЃљрЃюрЃў                              | рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў                          |
| ----------------------------------- | --------------------------------- |
| ­Ъћў рЃдрЃўрЃџрЃљрЃЎрЃќрЃћ рЃЊрЃљрЃЎрЃџрЃўрЃЎрЃћрЃЉрЃўрЃА рЃљрЃЏрЃЮрЃфрЃюрЃЮрЃЉрЃљ      | Alert рЃцрЃљрЃюрЃ»рЃљрЃарЃљ, рЃАрЃБрЃарЃљрЃЌрЃўрЃА рЃерЃћрЃфрЃЋрЃџрЃљ     |
| ­Ъћё рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃўрЃА рЃќрЃЮрЃЏрЃўрЃА рЃљрЃю рЃцрЃћрЃарЃўрЃА рЃерЃћрЃфрЃЋрЃџрЃљ  | рЃЊрЃўрЃЋрЃўрЃА рЃњрЃљрЃЊрЃўрЃЊрЃћрЃЉрЃљ                    |
| ­ЪДа рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃЦрЃЏрЃћрЃЊрЃћрЃЉрЃљрЃќрЃћ рЃарЃћрЃљрЃњрЃўрЃарЃћрЃЉрЃљ | рЃцрЃЮрЃарЃЏрЃўрЃА рЃерЃћрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃљ, рЃЏрЃљрЃБрЃАрЃўрЃА рЃЏрЃЮрЃФрЃарЃљрЃЮрЃЉрЃљ |
| ­ЪЊд рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃЏрЃўрЃдрЃћрЃЉрЃљ/рЃњрЃљрЃњрЃќрЃљрЃЋрЃюрЃљ      | рЃЕрЃљрЃбрЃў, рЃљрЃЏрЃўрЃюрЃЊрЃўрЃА рЃърЃарЃЮрЃњрЃюрЃЮрЃќрЃўрЃА рЃгрЃљрЃЎрЃўрЃЌрЃ«рЃЋрЃљ  |

---

### ­Ъј» **рЃерЃћрЃ»рЃљрЃЏрЃћрЃЉрЃљ:**

**JavaScript рЃЋрЃћрЃЉрЃњрЃЋрЃћрЃарЃЊрЃА "рЃљрЃфрЃЮрЃфрЃ«рЃџрЃћрЃЉрЃА"** Рђћ рЃўрЃњрЃў рЃърЃљрЃАрЃБрЃ«рЃЮрЃЉрЃА рЃЏрЃЮрЃЦрЃЏрЃћрЃЊрЃћрЃЉрЃћрЃЉрЃА, рЃфрЃЋрЃџрЃўрЃА рЃерЃўрЃюрЃљрЃљрЃарЃАрЃА рЃћрЃЎрЃарЃљрЃюрЃќрЃћ, рЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃА рЃЊрЃљ рЃљ.рЃе.

рЃњрЃўрЃюрЃЊрЃљ рЃљрЃњрЃўрЃ«рЃАрЃюрЃљ рЃњрЃљрЃюрЃАрЃ«рЃЋрЃљрЃЋрЃћрЃЉрЃљ HTML, CSS рЃЊрЃљ JavaScript-рЃА рЃерЃЮрЃарЃўрЃА?
