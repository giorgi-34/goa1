MERN Stack рЃљрЃарЃўрЃА рЃЋрЃћрЃЉ рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃћрЃЉрЃўрЃА рЃерЃћрЃАрЃљрЃЦрЃЏрЃюрЃћрЃџрЃў рЃЌрЃљрЃюрЃљрЃЏрЃћрЃЊрЃарЃЮрЃЋрЃћ рЃбрЃћрЃЦрЃюрЃЮрЃџрЃЮрЃњрЃўрЃћрЃЉрЃўрЃА рЃЎрЃЮрЃЏрЃърЃџрЃћрЃЦрЃбрЃў. рЃћрЃА рЃАрЃљрЃ«рЃћрЃџрЃгрЃЮрЃЊрЃћрЃЉрЃљ рЃерЃћрЃЊрЃњрЃћрЃЉрЃљ рЃЮрЃЌрЃ«рЃў рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў рЃбрЃћрЃЦрЃюрЃЮрЃџрЃЮрЃњрЃўрЃўрЃА рЃўрЃюрЃўрЃфрЃўрЃљрЃџрЃћрЃЉрЃўрЃАрЃњрЃљрЃю:

­Ъћи MERN = MongoDB + Express + React + Node.js
1. MongoDB РђЊ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЉрЃљрЃќрЃљ (Database)
рЃарЃЮрЃџрЃћ: рЃўрЃюрЃљрЃ«рЃљрЃЋрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃА (рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљ, рЃърЃЮрЃАрЃбрЃћрЃЉрЃў, рЃърЃарЃЮрЃЊрЃБрЃЦрЃбрЃћрЃЉрЃў рЃЊрЃљ рЃљ.рЃе.)

рЃбрЃўрЃърЃў: NoSQL рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЉрЃљрЃќрЃљ (рЃўрЃюрЃљрЃ«рЃљрЃЋрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃА JSON-рЃўрЃА рЃЏрЃАрЃњрЃљрЃЋрЃА рЃцрЃЮрЃарЃЏрЃљрЃбрЃерЃў РђЊ { key: value })

рЃЏрЃљрЃњ: {name: "рЃњрЃўрЃЮрЃарЃњрЃў", age: 20}

2. Express.js РђЊ рЃАрЃћрЃарЃЋрЃћрЃарЃўрЃА рЃЕрЃљрЃарЃЕрЃЮ (Backend Framework)
рЃарЃЮрЃџрЃћ: рЃЏрЃљрЃарЃЌрЃљрЃЋрЃА рЃАрЃћрЃарЃЋрЃћрЃарЃўрЃА рЃџрЃЮрЃњрЃўрЃЎрЃљрЃА рЃЊрЃљ рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃћрЃЉрЃА (routes, API)

рЃбрЃўрЃърЃў: Node.js-рЃўрЃА рЃЕрЃљрЃарЃЕрЃЮ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃљрЃЏрЃљрЃарЃбрЃўрЃЋрЃћрЃЉрЃА рЃАрЃћрЃарЃЋрЃћрЃарЃўрЃА рЃЊрЃљрЃгрЃћрЃарЃљрЃА

рЃЏрЃљрЃњ: /login, /get-products, /save-user

3. React.js РђЊ рЃцрЃарЃЮрЃюрЃб-рЃћрЃюрЃЊрЃўрЃА рЃЉрЃўрЃЉрЃџрЃўрЃЮрЃЌрЃћрЃЎрЃљ (Frontend Library)
рЃарЃЮрЃџрЃћ: рЃЦрЃЏрЃюрЃўрЃА рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃўрЃюрЃбрЃћрЃарЃцрЃћрЃўрЃАрЃА (UI), рЃарЃљрЃф рЃ«рЃћрЃЊрЃљрЃЋрЃА рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃў рЃЉрЃарЃљрЃБрЃќрЃћрЃарЃерЃў

рЃбрЃўрЃърЃў: JavaScript рЃЉрЃўрЃЉрЃџрЃўрЃЮрЃЌрЃћрЃЎрЃљ, рЃћрЃарЃЌрЃњрЃЋрЃћрЃарЃЊрЃўрЃљрЃюрЃў рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА (SPA)

рЃЏрЃљрЃњ: рЃдрЃўрЃџрЃљрЃЎрЃћрЃЉрЃў, рЃцрЃЮрЃарЃЏрЃћрЃЉрЃў, рЃАрЃўрЃћрЃЉрЃў, рЃњрЃЋрЃћрЃарЃЊрЃћрЃЉрЃўрЃА рЃерЃћрЃфрЃЋрЃџрЃљ рЃњрЃљрЃЊрЃљрЃбрЃЋрЃўрЃарЃЌрЃЋрЃўрЃА рЃњрЃљрЃарЃћрЃерЃћ

4. Node.js РђЊ рЃАрЃћрЃарЃЋрЃћрЃарЃўрЃА рЃњрЃљрЃарЃћрЃЏрЃЮ (JavaScript Runtime)
рЃарЃЮрЃџрЃћ: рЃўрЃДрЃћрЃюрЃћрЃЉрЃА JavaScript-рЃА рЃАрЃћрЃарЃЋрЃћрЃарЃўрЃА рЃЏрЃ«рЃљрЃарЃћрЃА (backend), рЃарЃўрЃЌрЃљрЃф рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃА рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃўрЃА рЃЏрЃБрЃерЃљрЃЮрЃЉрЃљрЃА

рЃбрЃўрЃърЃў: рЃћрЃ«рЃЏрЃљрЃарЃћрЃЉрЃљ Express-рЃА рЃЊрЃљ рЃАрЃ«рЃЋрЃљ рЃЏрЃЮрЃЊрЃБрЃџрЃћрЃЉрЃА рЃАрЃћрЃарЃЋрЃћрЃарЃќрЃћ рЃњрЃљрЃерЃЋрЃћрЃЉрЃљрЃерЃў

­ЪДЕ рЃарЃЮрЃњрЃЮрЃа рЃЏрЃБрЃерЃљрЃЮрЃЉрЃћрЃю рЃћрЃарЃЌрЃљрЃЊ?
React Рєњ рЃљрЃњрЃќрЃљрЃЋрЃюрЃўрЃА рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃљрЃА рЃАрЃћрЃарЃЋрЃћрЃарЃќрЃћ (рЃЏрЃљрЃњ. "рЃЏрЃЮрЃЏрЃљрЃгрЃЮрЃЊрЃћ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃћрЃЉрЃўрЃА рЃАрЃўрЃљ").

Express + Node.js Рєњ рЃўрЃдрЃћрЃЉрЃА рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃљрЃА рЃЊрЃљ рЃБрЃЎрЃљрЃЋрЃерЃўрЃарЃЊрЃћрЃЉрЃљ MongoDB-рЃА.

MongoDB Рєњ рЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃА рЃАрЃћрЃарЃЋрЃћрЃарЃќрЃћ.

Express + Node.js Рєњ рЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃА React-рЃерЃў.

React Рєњ рЃљрЃЕрЃЋрЃћрЃюрЃћрЃЉрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃА рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃА.

­ЪЊї MERN Stack-рЃўрЃА рЃБрЃърЃўрЃарЃљрЃбрЃћрЃАрЃЮрЃЉрЃћрЃЉрЃў:
рЃћрЃарЃЌрЃў рЃћрЃюрЃљ (JavaScript) Рђћ рЃцрЃарЃЮрЃюрЃб-рЃћрЃюрЃЊрЃќрЃћрЃф рЃЊрЃљ рЃЉрЃћрЃЦ-рЃћрЃюрЃЊрЃќрЃћрЃф

рЃАрЃгрЃарЃљрЃцрЃў рЃЊрЃљ рЃЏрЃЮрЃЦрЃюрЃўрЃџрЃў рЃњрЃљрЃюрЃЋрЃўрЃЌрЃљрЃарЃћрЃЉрЃљ

рЃЊрЃўрЃЊрЃў рЃЏрЃ«рЃљрЃарЃЊрЃљрЃГрЃћрЃарЃљ рЃЊрЃљ рЃарЃћрЃАрЃБрЃарЃАрЃћрЃЉрЃў

рЃњрЃўрЃюрЃЊрЃљ, рЃЏрЃљрЃарЃбрЃўрЃЋрЃў рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў рЃњрЃљрЃЕрЃЋрЃћрЃюрЃЮ MERN Stack рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃўрЃА рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃўрЃА?
