JavaScript-рЃўрЃА рЃЎрЃЮрЃЊрЃў рЃЋрЃћрЃЉрЃњрЃЋрЃћрЃарЃЊрЃќрЃћ рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃЊрЃљрЃўрЃгрЃћрЃарЃЮрЃА рЃарЃљрЃЏрЃЊрЃћрЃюрЃўрЃЏрЃћ рЃњрЃљрЃюрЃАрЃ«рЃЋрЃљрЃЋрЃћрЃЉрЃБрЃџ рЃљрЃЊрЃњрЃўрЃџрЃљрЃА рЃЊрЃљ рЃАрЃбрЃўрЃџрЃерЃў. рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџрЃА рЃЌрЃљрЃЋрЃўрЃАрЃў рЃЊрЃљрЃюрЃўрЃерЃюрЃБрЃџрЃћрЃЉрЃљ рЃЊрЃљ рЃАрЃўрЃбрЃБрЃљрЃфрЃўрЃљ рЃљрЃЦрЃЋрЃА. рЃЦрЃЋрЃћрЃЏрЃЮрЃЌ рЃЕрЃљрЃЏрЃЮрЃЋрЃгрЃћрЃа рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃЏрЃўрЃЊрЃњрЃЮрЃЏрЃћрЃЉрЃў:

---

### РюЁ **1. Inline JavaScript (рЃљрЃбрЃарЃўрЃЉрЃБрЃбрЃерЃў рЃЊрЃљрЃгрЃћрЃарЃўрЃџрЃў рЃЎрЃЮрЃЊрЃў)**

­ЪЊї **рЃљрЃдрЃгрЃћрЃарЃљ:** рЃЎрЃЮрЃЊрЃў рЃгрЃћрЃарЃўрЃљ рЃърЃўрЃарЃЊрЃљрЃърЃўрЃа HTML рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃўрЃА рЃљрЃбрЃарЃўрЃЉрЃБрЃбрЃерЃў Рђћ рЃЕрЃЋрЃћрЃБрЃџрЃћрЃЉрЃарЃўрЃЋ `onclick`, `onmouseover` рЃЊрЃљ рЃАрЃ«рЃЋрЃљ рЃЏрЃЮрЃЋрЃџрЃћрЃюрЃћрЃЉрЃерЃў.

­Ъћ╣ **рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў:**

```html
<button onclick="alert('рЃАрЃљрЃџрЃљрЃЏрЃў')">рЃЊрЃљрЃгрЃЎрЃљрЃърЃћ</button>
```

---

### РюЁ **2. Internal JavaScript (script рЃбрЃћрЃњрЃерЃў HTML рЃцрЃљрЃўрЃџрЃўрЃА рЃерЃўрЃњрЃюрЃўрЃЌ)**

­ЪЊї **рЃљрЃдрЃгрЃћрЃарЃљ:** рЃЎрЃЮрЃЊрЃў рЃўрЃгрЃћрЃарЃћрЃЉрЃљ `<script>` рЃбрЃћрЃњрЃерЃў рЃўрЃЏрЃљрЃЋрЃћ HTML рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃўрЃА рЃЌрЃљрЃЋрЃерЃў рЃљрЃю рЃЉрЃЮрЃџрЃЮрЃерЃў.

­Ъћ╣ **рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў:**

```html
<script>
  function showMessage() {
    alert("рЃњрЃљрЃЏрЃљрЃарЃ»рЃЮрЃЉрЃљ!");
  }
</script>

<button onclick="showMessage()">рЃюрЃљрЃ«рЃћ рЃерЃћрЃбрЃДрЃЮрЃЉрЃўрЃюрЃћрЃЉрЃљ</button>
```

---

### РюЁ **3. External JavaScript (рЃњрЃљрЃарЃћ рЃцрЃљрЃўрЃџрЃўрЃЊрЃљрЃю рЃЏрЃўрЃЉрЃЏрЃБрЃџрЃў)**

­ЪЊї **рЃљрЃдрЃгрЃћрЃарЃљ:** JavaScript рЃЎрЃЮрЃЊрЃў рЃўрЃюрЃљрЃ«рЃћрЃЉрЃљ рЃфрЃљрЃџрЃЎрЃћ `.js` рЃцрЃљрЃўрЃџрЃерЃў рЃЊрЃљ HTML-рЃерЃў рЃћрЃарЃЌрЃЊрЃћрЃЉрЃљ `<script src="script.js"></script>`-рЃўрЃЌ.

­Ъћ╣ **HTML рЃцрЃљрЃўрЃџрЃў:**

```html
<script src="script.js"></script>
```

­Ъћ╣ **script.js рЃцрЃљрЃўрЃџрЃў:**

```javascript
function showName() {
  alert("рЃњрЃўрЃЮрЃарЃњрЃў рЃЦрЃљрЃЋрЃЌрЃљрЃарЃљрЃФрЃћ");
}
```

---

### ­ЪДа **рЃерЃћрЃ»рЃљрЃЏрЃћрЃЉрЃљ:**

| # | рЃЏрЃўрЃЊрЃњрЃЮрЃЏрЃљ             | рЃцрЃЮрЃарЃЏрЃљ                          | рЃЎрЃЮрЃЏрЃћрЃюрЃбрЃљрЃарЃў                                         |
| - | ------------------- | ------------------------------ | ------------------------------------------------- |
| 1 | Inline JavaScript   | `onclick="..."`                | рЃЏрЃљрЃарЃбрЃўрЃЋрЃў, рЃЏрЃљрЃњрЃарЃљрЃЏ рЃљрЃарЃљрЃБрЃЎрЃЋрЃћ рЃБрЃЎрЃћрЃЌрЃћрЃАрЃў рЃърЃарЃљрЃЦрЃбрЃўрЃЎрЃљ          |
| 2 | Internal JavaScript | `<script>...</script>` HTML-рЃерЃў | рЃАрЃљрЃАрЃгрЃљрЃЋрЃџрЃЮрЃЊ рЃЊрЃљ рЃЏрЃфрЃўрЃарЃћ рЃърЃарЃЮрЃћрЃЦрЃбрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА рЃЎрЃљрЃарЃњрЃў           |
| 3 | External JavaScript | `<script src="...">`           | рЃърЃарЃЮрЃцрЃћрЃАрЃўрЃЮрЃюрЃљрЃџрЃБрЃарЃў рЃЊрЃљ рЃАрЃБрЃцрЃЌрЃљ рЃЎрЃЮрЃЊрЃўрЃАрЃЌрЃЋрЃўрЃА рЃарЃћрЃЎрЃЮрЃЏрЃћрЃюрЃЊрЃўрЃарЃћрЃЉрЃБрЃџрЃў |

---

рЃњрЃўрЃюрЃЊрЃљ рЃљрЃњрЃўрЃ«рЃАрЃюрЃљ, рЃарЃљрЃбрЃЮрЃЏ рЃљрЃарЃўрЃА **external JS** рЃАрЃљрЃБрЃЎрЃћрЃЌрЃћрЃАрЃЮ рЃЋрЃљрЃарЃўрЃљрЃюрЃбрЃў рЃЊрЃўрЃЊ рЃърЃарЃЮрЃћрЃЦрЃбрЃћрЃЉрЃерЃў?
